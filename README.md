Learn Basics > Cover > File Systems,cd,ls,cat,mkdir,rm,cp,mv,permissions (Chmod , Chown)
-----------------------------------------------------------------------------------------
**File and directory navigation:**
ls: List directory contents.
cd: Change directory.
pwd: Print the current working directory.
mkdir: Create a new directory.
rmdir or rm -r: Remove a directory.
touch: Create a new, empty file.

**File and directory manipulation:**
cp: Copy files.
mv: Move or rename files.
rm: Delete files.
Viewing and editing files:
cat: Concatenate and display file content.
less or more: View files page by page.
head or tail: Display the beginning or end of a file.
nano or vim: Text editors.

**Permissions and ownership:**
chmod: Change file permissions.
chown: Change file ownership.

**Networking:**
ping: Test network connectivity.
ssh: Securely connect to a remote machine.
scp: Securely copy files between hosts.
netstat: Display network connections.
